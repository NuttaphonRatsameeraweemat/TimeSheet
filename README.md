# TimeSheet
TimeSheet Api System

## Description

The TimeSheet Api Project is my example setup project with .net core 

## Prerequisites
You need to install these tools first in order to develop this project:
 - .NET Core 2.2

## Installation

```bash
dotnet restore
```

## Usage

```bash
dotnet run
```
